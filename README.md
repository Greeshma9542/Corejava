# Shopping Mall Management System

## Overview
This project is a console-based application developed using Core Java to manage shopping mall operations such as customer details, shop management, product handling, and billing.

## Technologies Used
- Core Java
- OOP Concepts
- Collections (ArrayList)
- File Handling (if used)

## Features
- Manage customer details
- Manage shops and products
- Generate bills
- Search functionality
- Menu-driven interface

## How to Run
1. Install Java (JDK 8 or above)
2. Compile the Java files
3. Run the main class

- Add GUI (Swing / JavaFX)
